# Buscar Usuários – Projeto de Estudos

Projeto simples desenvolvido com **HTML, CSS e JavaScript**, com o objetivo de praticar o consumo de uma **API externa**, manipulação do DOM e estilização de interfaces.

A aplicação permite buscar usuários pelo **ID** e exibir suas informações na tela.

---

## Funcionalidades

- Buscar usuários pelo **ID (1 a 30)**
- Consumo de API externa usando `fetch`
- Exibição dinâmica dos dados do usuário
- Validação de campo vazio
- Mensagem de erro para usuário não encontrado
- Interface moderna e responsiva

---

## Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**
- **API DummyJSON**
- **Google Fonts (Poppins)**

---

## API Utilizada

Os dados dos usuários são obtidos através da API pública:
https://dummyjson.com/users
